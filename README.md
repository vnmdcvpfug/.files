### Description
- Document viewer - [zathura](https://github.com/pwmt/zathura)
- Music player - [inori](https://github.com/eshrh/inori)
- Terminal - [Wez's Terminal](https://github.com/wezterm/wezterm)
- Terminal status line - [tmux](https://github.com/tmux/tmux)
- Text editor - [neovim](https://github.com/neovim/neovim)
- Window manager - [Hyprland](https://github.com/hyprwm/Hyprland)

### Installation
```
# Install git if it's not already installed
sudo pacman -S git

# Go to $HOME
cd ~

# Clone this repository
git clone https://github.com/vnmdcvpfug/.files.git

# Go to the repository directory
cd dotfiles

# Run the installation script, then follow the on-screen instructions
./install.sh
```
